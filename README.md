# A/B Testing Product Optimization Analysis

## Project Overview
This project analyzes wheather a redesignd homepage experience improved user conversion compare to the existing homepage. The goal was to determine wheather the new experience should be rolled out based on conversion lift and statistical significance.

## Business Problem
Conversion rates on a key product page were lower than expected. The business wanted to understand wheather a new page layout would improve user engagement and purchases. 

## Objectives
Evaluate the performance of a control group versus a test group and provide a reccomendation on whether to launch the new design.

## Tools Used
- Python
- Pandas
- NumPy
- SciPy
- Power BI
- Excel/CSV
- GitHub

## Dataset
The dataset includes sample user level experiment data with the following fields:
- user_id
- experiment_group
- converted
- session_duration
- device_type
- traffic_source

## Methodology
1. Cleaned and Validated the dataset.
2. Compared conversion rates between control and test groups.
3. Calculated conversion lift.
4. Performed a statistical significance test.
5. Segmented results by device type and traffic source.
6. Developed a business reccomendation based on results.

## Key Metrics
- Conversion Rate
- Absolute Lift
- Relative Lift
- Statistical Significance
- Segment Performance

## Results Summary
The test group showed a higher conversion rate than the control group. The analysis evaluated wheather the lift was statistically significant and wheather the redesign should be rolled out.
